## 서비스 개요

**발자국**은 반려견과 함께 떠나는 여행을 위한 AI 여행 플래너 앱입니다.
사용자가 반려견 정보를 입력하면, 애견 전용 LLM 어시스턴트 **코코(Coco)** 가
반려견의 정보(체중·크기 등)에 맞춰 입장 가능한 문화시설과 장소만 필터링하여
맞춤형 여행 계획을 제안합니다.

### 핵심 기능
- 🐾 반려견 정보 기반 개인화 필터링 (입장 가능 장소 자동 선별)
- 🗣️ 음성 호출 가능한 애견 전용 LLM "코코" (Wake Word: "코코야~")
- 🏛️ 문화체육관광부 산하 문화시설 데이터 연동
- 🚗 이동수단별 맞춤 경로 및 경유지 추천
- 🏨 반려견 동반 가능 숙소 자동 추천
- 🚨 응급상황 시 주변 동물병원 자동 안내

## 시나리오 1 · 전주 문화유산 여행 (대표)

### "김민주 × 체리의 전주 문화유산 여행"

**배경** : 서울 거주 30대 직장인 김민주, 소형견 체리(말티즈, 5kg)와 함께 전주 1박 2일 여행

---

#### Day 1 흐름

```
[서울 출발]
김민주 : "코코야, 전주로 여행 갈 거야"
코코   : "이동 수단은 어떻게 하실 건가요?"
김민주 : "자차로 갈 거야"
코코   : "가시는 길에 체리와 함께 들르기 좋은 곳 추천해드릴게요!"

        ↓ 경유지 추천 (자차 루트)

🗺️ 서울 → [수원 광교호수공원 · 반려견 동반 가능]
         → [용인 에버랜드 인근 펫파크]
         → 전주 도착

        ↓ 전주 도착 후

🏯 전주 한옥마을 (문화시설 · 반려견 입장 가능 구역 필터링)
🎨 전주 국립무형유산원 야외 공간 (문화부 산하 시설)
🍃 전주 덕진공원 (반려견 산책 가능)

        ↓ 숙박

🏨 체리 동반 가능 한옥 게스트하우스 자동 추천
```

#### Day 2 흐름

```
🏛️ 전주 국립전주박물관 (야외 조각공원 · 반려견 동반 가능)
🎭 전주 한국전통문화전당 (문화부 산하 · 야외행사 시 입장 가능)
🍜 반려견 동반 가능 전주 현지 맛집 추천
```

#### 핵심 기능 포인트
- 자차 경유지 자동 추천
- 문화부 산하 시설 중 반려견 허용 구역 필터링
- 체리 체중(5kg) 기준 입장 가능 여부 자동 판단

## 시나리오 2 · 경주 역사문화 탐방

### "이준혁 × 몽이의 경주 역사문화 탐방"

**배경** : 부산 거주 40대 가족(부부 + 자녀 1명), 중형견 몽이(진돗개, 18kg)와 경주 당일치기

```
[핵심 갈등 포인트]
몽이는 18kg 중형견 → 입장 제한 시설이 많음
→ 코코가 몽이 체중 기준으로 엄격하게 필터링
```

---

#### 여행 흐름

```
이준혁 : "코코야, 경주 역사유적지 가보고 싶어"
코코   : "몽이는 18kg라 입장 제한이 있는 곳이 있어요.
          몽이와 함께 갈 수 있는 곳만 보여드릴게요!"

        ↓ 문화시설 필터링 결과

✅ 입장 가능
🏛️ 경주 동궁과 월지 (안압지) · 야외 공원 구역
🗿 경주 대릉원 · 야외 능역 산책 가능
🌿 경주 보문관광단지 · 반려견 동반 가능

❌ 입장 불가 (코코가 자동 제외)
🚫 경주 국립박물관 실내
🚫 불국사 경내

        ↓ 코코 대안 제시

코코 : "불국사 실내는 어렵지만,
        토함산 자락 야외 트레킹 코스는
        몽이와 함께 가능해요! 추가할까요?"
```

#### 핵심 기능 포인트
- 반려견 체중/크기별 입장 가능 시설 자동 필터링
- 입장 불가 시 **대안 장소 자동 제안** 기능
- 가족 단위 여행 (아이 + 반려견 동반) 동선 최적화
- 문화재 보호구역 내 반려견 규정 정보 제공

## 시나리오 3 · 제주 문화예술 여행

### "박서연 × 두부의 제주 문화예술 여행"

**배경** : 서울 거주 20대 커플, 소형견 두부(포메라니안, 3kg)와 제주 3박 4일

```
[핵심 특징]
- 비행기 이동 (항공사별 반려견 기내 반입 규정 안내)
- 제주 문화예술 시설 중심
- 렌터카 이용 (반려견 동반 렌터카 필터링)
```

---

#### 여행 흐름

```
박서연 : "코코야, 두부랑 제주 3박 4일 가려고"
코코   : "이동 수단은요?"
박서연 : "비행기"
코코   : "두부(3kg)는 기내 반입 가능해요! ✅
          제주항공 / 티웨이 기내 반입 조건 안내해드릴게요.
          이동장 규격 : 가로 40 × 세로 25 × 높이 20cm 이하"

        ↓ 제주 도착 후

코코 : "반려견 동반 가능한 렌터카 업체 3곳 추천해드릴게요"

        ↓ 일정 구성

[Day 1]
🎨 제주도립미술관 · 야외 조각공원 (반려견 동반 가능)
🌊 협재해수욕장 (반려견 입장 가능 시간대 안내)

[Day 2]
🏛️ 제주 국립제주박물관 야외 공간
🌿 한림공원 일부 구역 (반려견 동반 가능 구역 필터링)
🎭 제주 문화예술진흥원 야외 행사 (시즌별 안내)

[Day 3]
🖼️ 이중섭미술관 야외 공간 (문화부 연계 시설)
🐴 산굼부리 (반려견 동반 가능 야외 명소)

[Day 4]
🛍️ 반려견 동반 카페 / 제주 공항 이동
코코 : "귀가 시 두부 스트레스 최소화 팁도 알려드릴게요 🐾"
```

#### 핵심 기능 포인트
- **항공사별 반려견 기내 반입 규정** 자동 안내
- 반려견 동반 렌터카 필터링
- 제주 문화예술 시설(미술관, 박물관) 반려견 허용 구역 세분화
- 여행 중 **반려견 응급 동물병원** 위치 자동 표시
- 시즌별 야외 문화행사 연동 추천

## 시나리오 4 · 남해안 문화 호핑 여행

### "최동현 × 콩이의 남해안 문화 호핑 여행"

**배경** : 서울 거주 30대 프리랜서 최동현, 소형견 콩이(비숑, 6kg)와 남해안 4박 5일 **멀티 거점 여행**

```
[핵심 특징]
- 매일 숙소를 옮기는 "호핑(Hopping)" 스타일
- 자차 이용 · 남해안 문화도시 순환
- 콩이 동반 가능 숙소를 거점별로 각각 추천
```

---

#### 여행 도입

```
최동현 : "코코야, 남해안 쭉 돌면서 여행하고 싶어.
          숙소도 매일 바꿀 거야"

코코   : "호핑 여행이군요! 🐾
          매 거점마다 콩이와 함께 숙박 가능한
          숙소를 각각 추천해드릴게요.
          체크아웃 / 체크인 동선도 최적화해드릴게요!"
```

#### 거점별 일정

```
[Day 1 · 거점 : 여수]
─────────────────────────────
이동 : 서울 → 여수 (자차 · 약 4시간)

🏛️ 여수 예울마루 (문화부 연계 · 야외공연장 반려견 동반 가능)
🌊 여수 이순신광장 야외 산책
🎨 여수 국동항 벽화마을

🏨 숙소 1 : 여수 돌산도 펫 동반 풀빌라
            (콩이 6kg · 소형견 가능 확인 ✅)

코코 알림 : "내일 체크아웃 후 순천까지 30분이에요.
             콩이 이동 스트레스 최소 구간입니다 🐾"
```

```
[Day 2 · 거점 : 순천]
─────────────────────────────
이동 : 여수 → 순천 (자차 · 약 30분)

코코 : "체크아웃 전에 콩이 산책할 곳 주변에 있어요!"

🌿 순천만 국가정원 (반려견 동반 가능 구역 필터링)
🏛️ 순천 문화의전당 야외 공간
🎭 순천 드라마 촬영장 (야외 · 반려견 입장 가능)

🏨 숙소 2 : 순천 조례호수공원 인근 펫 동반 호텔
            (콩이 6kg · 소형견 가능 확인 ✅)

코코 알림 : "내일 보성까지 1시간이에요.
             중간에 콩이 휴게 스팟 1곳 추천해드릴게요!"
```

```
[Day 3 · 거점 : 보성 · 담양]
─────────────────────────────
이동 : 순천 → 보성 → 담양 (자차 · 약 1시간 30분)

🍵 보성 녹차밭 (야외 · 반려견 동반 가능)
🎋 담양 죽녹원 (반려견 동반 가능 구역 안내)
🏛️ 담양 한국대나무박물관 야외 공간

🏨 숙소 3 : 담양 펫 동반 한옥 스테이
            (콩이 6kg · 소형견 가능 확인 ✅)

코코 알림 : "담양 한옥 스테이는 마당이 넓어서
             콩이가 뛰어놀기 좋아요 🐾"
```

```
[Day 4 · 거점 : 통영]
─────────────────────────────
이동 : 담양 → 통영 (자차 · 약 2시간)

🎨 통영 국제음악당 야외 공간 (문화부 연계 시설)
🖼️ 전혁림미술관 야외 공간
🌊 통영 강구안 문화마당 (야외 · 반려견 산책 가능)
🚢 통영 한산도 여객선
    코코 : "여객선 반려견 탑승 규정 안내해드릴게요 🐾"

🏨 숙소 4 : 통영 미륵도 오션뷰 펫 동반 펜션
            (콩이 6kg · 소형견 가능 확인 ✅)
```

```
[Day 5 · 귀가]
─────────────────────────────
이동 : 통영 → 서울 (자차 · 약 4시간)

코코 : "서울까지 4시간이에요.
        중간 휴게소 중 반려견 동반 가능한
        휴게소 2곳 경유지로 추가해드릴까요?"

✅ 함양 휴게소 · 반려견 산책공간 있음
✅ 신탄진 휴게소 · 반려견 동반 가능
```

#### 호핑 여행 전용 핵심 기능

```
1. 거점별 숙소 일괄 추천
   → 전체 여정 한눈에 보기
   → 각 거점 숙소 콩이 입장 가능 여부 일괄 확인

2. 체크아웃 → 이동 → 체크인 동선 최적화
   → 콩이 이동 시간 최소화 루트 자동 계산
   → 이동 거리 2시간 초과 시 중간 휴식 스팟 자동 추가

3. 숙소 간 거리 기반 일정 자동 조정
   → 이동이 긴 날은 관광 일정 축소
   → 이동이 짧은 날은 문화시설 추가 추천

4. 매일 아침 코코 알림
   → "오늘 체크아웃 시간 · 다음 숙소까지 거리 ·
      콩이 산책 추천 장소" 자동 브리핑

5. 반려견 이동 스트레스 지수 안내
   → 하루 이동 시간이 3시간 초과 시
      코코가 경고 및 일정 조정 제안
```

## 시나리오 5 · 여행 중 응급상황 대응

### "김민주 × 체리의 전주 여행 중 응급상황"

**배경** : 전주 한옥마을 여행 중 체리가 갑자기 이상 증상을 보임

---

#### 🚨 응급 감지 흐름

```
김민주 : "코코야, 체리가 아파"

코코 : 🚨 즉시 응급모드 전환
        "체리 괜찮아요?! 지금 바로
         주변 운영중인 동물병원 찾아드릴게요!"

        ↓ 동시에 자동 실행

✅ GPS 현재 위치 감지
✅ 현재 시간 기준 운영중인 병원만 필터링
✅ 24시간 응급 동물병원 우선 노출
✅ 거리순 / 운영시간순 정렬
```

#### 💬 코코 대화 흐름

```
코코 : "체리 상태를 조금 더 알려주실 수 있어요?
        빠르게 맞는 병원을 찾아드릴게요 🏥"

        ↓ 증상 키워드 자동 감지
```

## 해커톤 어필 전략

### ✅ 문화체육관광부 연계
- 국립박물관, 무형유산원, 문화전당, 예울마루, 국제음악당 등
  문화부 산하·연계 시설 데이터를 직접 활용
- 전국 문화시설의 반려견 동반 가능 구역 DB 구축 제안

### ✅ 사회적 가치
- 반려인 1500만 시대, 문화시설 접근성 향상
- 단순 여행을 넘어 반려동물 응급 케어까지 포함한 종합 플랫폼

### ✅ 기술적 차별점
- 반려견 정보(체중·크기) 기반 개인화 필터링
- 애견 전용 LLM "코코" 적용 (음성 호출 Wake Word 지원)
- 실시간 위치·운영시간 데이터 연동 (응급 동물병원 안내)

### ✅ 다양한 여행 스타일 커버
- 당일치기 / 거점형 / 휴양형 / 호핑형 / 응급 대응형
- 모든 유형의 반려인 사용자 포괄

### ✅ 확장 가능성
- 전국 문화시설 반려견 허용 DB 구축
- 항공·렌터카·숙소·동물병원 등 파트너십 연계 확장
- 반려동물 복지 중심 설계로 지속 가능한 서비스 포지셔닝
