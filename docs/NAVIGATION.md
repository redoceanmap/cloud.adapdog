# 발자국 — 화면 구조 / 내비게이션 (NAVIGATION.md)

> 통합본(56화면) 기준. 프론트 라우팅·화면 배치의 단일 기준.
> `발자국_PROJECT_CONTEXT.md`(기능·데이터·AI)와 함께 읽는다.

## 전역 규칙
- **하단 탭 4개**: 플래너 / 둘러보기 / 커뮤니티 / 내 강아지. 그 외는 탭으로 만들지 않는다.
- **응급은 탭이 아니다.** 전 화면 우하단 고정 FAB(코랄) → 응급 AI 증상 체크 + 가까운 병원.
- **지도는 탭이 아니다.** 플래너·둘러보기 결과 화면의 "리스트 ↔ 지도" 토글.
- **발자국 지도(추억) ≠ 길찾기 지도(기능)**: 전자는 [내 강아지], 후자는 [플래너] 결과.
- 강아지 표현은 전부 3D(3D 실사 + 3D 마스코트). 평면 2D 일러스트 사용 금지.
- 데이터 카드마다 출처 태그. 동물병원=행정안전부, 둘레길=두루누비.

## 진입 / 온보딩 (탭 진입 전 1회) — A·B 섹션
A1 시작 → A2 시작 방법(사진/직접) → A3 사진 업로드 → A4 견종 인식 결과(ML, 일치율·품종 특성 칩·설명) → A5 프로필 입력 → A6 프로필 완성 → B1 페르소나(실사) / B3 페르소나 실사 히어로 → 메인([플래너] 탭).
- B2 특징 편집(활동량·사교성·더위 민감도, 메뉴에서만 선택·직접입력 비활성)은 [내 강아지] 프로필에서 진입.

## 탭 1 · 플래너 (탐색) — C 섹션
- 흐름: C1 프롬프트 입력 → C2 추천 코스 리스트(리스트↔지도 토글) → C3 시설 상세·입장 판정 → [여행 시작] → 여행 진행(D).
- 시설 상세 구성: C4 배지 시스템(목줄/이동장/실내/추가요금/동반 크기) · C5 맞춤 입장 판정 · C6 발자국 지도(동선) · C7/C8 이동약자 배려 토글(OFF 전체 / ON 배려 시설만) · C9 배려 배지 가이드 · C10 식당 예약(mock) · C11 숙소 예약(mock) · C12 닮은 친구 추천.
- endpoint 힌트: `POST /trips/plan`, `GET /places`, `GET /places/:id`, `POST /places/cards`.

## 탭 1 연장 · 여행 진행 — D 섹션
D1 케어 타임라인 · D2 케어 알림 설정 · D3 QR 스캔(현장 안내) · D4 관광지 오디오 가이드 · D5 오디오 컴포넌트 상태 · D6 응급 진입 → D7 증상 첨부 → D8 분석 중 → D9 AI 결과(참고용·진단 아님·반드시 수의사·응급 시 즉시 병원) · D10 가까운 병원 리스트 · D11 병원 상세(영업·영업시간·전화). 응급은 전역 FAB로도 진입.

## 탭 2 · 둘러보기 (문화·여행 데이터 허브) — E 섹션
E1 월 캘린더 뷰(동반 가능일) · E2 축제 리스트·카드 · E3 걷기 코스 상세(둘레길) · E4 문화시설 스탬프 컬렉션. 각 목록 리스트↔지도 토글. QR(D3)은 오디오·스탬프 인증에서 호출. "공공 문화데이터" 쇼케이스.

## 탭 3 · 커뮤니티 — H1~H3
H1 커뮤니티 홈(코스 디스커버리) · H2 코스 후기 상세(그대로 담기) · H3 발자국 클립(강아지 시점 브이로그). 남의 코스·기록 발견 전용.

## 탭 4 · 내 강아지 (프로필 허브) — B2 · F · G · H4 · H5
시각 계층(위→아래):
1. 히어로: 3D 강아지(마스코트/실사) + 이름·견종 + 핵심 칩.
2. 탭 전환: [프로필(지갑)] · [꾸미기].
3. (지갑) 빠른 액션 행: 응급 · 케어 알림 · 스탬프 현황.
4. (지갑) 기본 정보 카드 + B2 특징 편집 진입.
5. (지갑) 추억: H5 연말 결산 스토리 · 발자국 지도 · G(브이로그) · 내 공유 코스 · H4 발자국 패스포트.
6. (지갑) 케어 알림 설정.
7. (꾸미기) F1 템플릿 갤러리(전통 한복·문양·여행지·축제·계절) → F2 적용 중 → F3 결과 카드 → F4 테마 갤러리. 굿즈/3D 피규어: F5 사진 → F6 스타일 → F7 생성 중 → F8 3D 리빌(360) → F9 커스터마이즈 → F10 저장·공유·굿즈(mock).

## 브이로그 — G 섹션 (커뮤니티/내 강아지에서 진입)
G1 코스 선택 → G2 클립 선택 → G3 톤 → G4 생성 중 → G5 완성(강아지 시점·세로 풀스크린·자막·공유).

## 3D 자산 규칙
사진 업로드(A3)·견종 인식(A4)·페르소나(B1/B3)·피규어(F5~F10)·프로필/결산 썸네일은 "업로드한 그 강아지"의 3D 실사 또는 3D 마스코트로 표시. 데모는 체리 사전 제작 에셋 사용. 생성은 포트 뒤 어댑터(`adapters/outbound/ml/`/외부 3D API), 실패 시 사전 에셋으로 degrade.
